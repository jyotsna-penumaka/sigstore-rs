hyperx

Hyper is the low-level HTTP implementation for Rust. Hyperx is an extraction of the hyper 0.11 typed header module, with minimized dependencies, for continued use with hyper 0.12 or later, where this module was removed in preference to the byte-oriented http::header module.

See the header module for more details.
