The default signature verification algorithm used by Sigstore. Sigstore relies on NIST P-256 NIST P-256 is a Weierstrass curve specified in FIPS 186-4: Digital Signature Standard (DSS). Also known as prime256v1 (ANSI X9.62) and secp256r1 (SECG)
